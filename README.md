# X Clone

A web-based clone of the X (formerly Twitter) timeline interface, built with HTML, CSS, and JavaScript. This project replicates the look and feel of X, including a timeline feed, tweet composer, trending topics, and suggested follows.

---

## Features

- Responsive layout for desktop and mobile
- Sticky header with tabs for "For you" and "Following"
- Tweet composer with character count and posting restrictions (280 characters)
- Static and dynamic tweet feed including images and video posts
- Sidebar with search, trends, news, and follow suggestions
- Interactive buttons for likes, comments, shares, retweets, and bookmarks
- Post modal for creating posts with proper accessibility support
- Accessible design using ARIA labels and semantic HTML elements
